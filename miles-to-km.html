When to use this miles to kilometres converter

Miles are still common on road signs, running plans, and travel details in some countries, while kilometres are standard elsewhere. This converter helps you switch an imperial distance into a metric value you can use more easily.

Miles to KM FAQ

How many kilometres are in one mile?

One mile equals about 1.60934 kilometres. Multiply miles by 1.60934 to estimate the metric distance.

Is this useful for race training?

Yes. It helps when training plans, app data, or event descriptions mix miles and kilometres.

Can I use it for international travel?

Absolutely. It is useful when maps, route planners, or road signs use kilometres instead of miles.
