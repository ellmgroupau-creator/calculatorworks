Formula

new value = original value × (1 - percentage ÷ 100)

Example: 120 decreased by 15% = 102.000

FAQ

When should I use percentage decrease?

Use it when you know the starting value and the reduction rate and want the reduced amount immediately.

How is this different from percent change?

Percent change compares an old value and a new value, while percentage decrease applies a chosen reduction to one starting number.

Can it help with discounts?

Yes. It works well for discount maths, markdowns, and budget-reduction scenarios.
