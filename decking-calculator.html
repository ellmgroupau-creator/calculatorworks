Decking Calculator

Estimate the number of decking boards needed for a rectangular deck area.

Result
Enter values to begin

Formula

boards needed = deck area × (1 + waste %) ÷ coverage per board.

Worked example

A 24 m² deck using boards covering 0.32 m² each with 10% waste needs about 83 boards.

How to use this calculator

Use the actual board dimensions and include waste for cuts, joins, layout direction and damaged boards.
