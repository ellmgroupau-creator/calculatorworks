Calculadora de porcentajes

Introduce un porcentaje y un número para calcular X% de Y.

Resultado
Introduce un valor para empezar

Fórmula

resultado = (porcentaje ÷ 100) × número

Ejemplo: 15% de 200 = 30.000

Ejemplos resueltos

10% de 50 = 5.000

15% de 200 = 30.000

22,5% de 80 = 18.000

Usos comunes

Esta herramienta es útil para impuestos, propinas, descuentos, comisiones y comprobaciones rápidas de cálculo mental.

¿Cómo calculo X% de un número?

Multiplica el número por el porcentaje y divide entre 100.

¿Cuándo debo usar esta herramienta?

Cuando ya conoces el porcentaje y el número base.

¿Sirve para descuentos?
